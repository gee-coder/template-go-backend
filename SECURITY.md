# 安全说明

感谢你帮助提升模板仓库的安全性。

## 1. 适用范围

本仓库是基础模板，以下问题建议优先通过私下方式反馈：

- 默认密钥或示例配置存在误用风险
- 鉴权、权限、Token、刷新机制存在逻辑漏洞
- 敏感信息泄露
- 容器或部署默认配置存在高风险问题

## 2. 不建议公开披露的内容

- 可直接利用的漏洞细节
- 可访问生产环境的密钥、Token、账号
- 可复现的攻击脚本

## 3. 推荐反馈方式

- 先准备漏洞描述、影响范围、复现步骤、修复建议
- 不要在公开 issue 中贴出敏感信息
- 如必须先提 issue，请只描述现象，不公开利用细节

## 4. 修复策略

维护者会在确认后尽快修复，并在必要时补充 README、示例配置或代码注释，避免后续使用者继续踩坑。
